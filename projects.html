SICA for Music

Subjectively Interesting Component Analysis is a method for finding interesting projects given some prior knowledge.

Example Dependent Cost-Sensitive Learning

Cost-sensitive learning framework where in one of the experiments, popularity is used in the cost function. The idea is that more popular songs will be searched more frequently therefore one would want to perform better on these songs. Published as a workshop paper in ECML LIDTA 2018.

Album Cover Generation from Genre Tags

Published as a conference paper in 10th International Workshop on Machine Learning and Music. Work completed as part of my undergraduate degree.
Using generative adversarial network to generate album cover images given the albums genre. The similarity between real album covers and generated images was measured using MS-SSIM.

Normalised Laplacian Pyramid Distance for Music

Using Normalised Laplacian Pyramid Distance to measure quality and simiarlity between chromagrams of musical tracks.
